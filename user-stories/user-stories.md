# User Stories - Backlog Recovery

## Registrar dueño de mascota
Como recepcionista, quiero registrar a un nuevo dueño en el sistema, con el fin de poder asociar correctamente a sus mascotas.  
Fuente: OwnerController.java

## Buscar dueños de mascotas
Como recepcionista, quiero buscar a los dueños por apellido, para poder acceder de manera rápida a la información relacionada a cada uno.  
Fuente: OwnerController.java

## Ver información de un dueño
Como recepcionista, quiero visualizar la información de un dueño, para consultar sus mascotas y el historial de visitas registradas.  
Fuente: OwnerController.java

## Agregar mascota a un dueño
Como recepcionista, quiero agregar una mascota a un dueño ya existente, para mantener actualizada la información del sistema.  
Fuente: PetController.java

## Actualizar información de una mascota
Como recepcionista, quiero actualizar los datos de una mascota, para asegurar que la información sea correcta.  
Fuente: PetController.java

## Registrar una visita médica
Como recepcionista, quiero registrar una visita médica para una mascota, con el fin de llevar un control.  
Fuente: VisitController.java

## Ver veterinarios
Como administrador, quiero visualizar el listado de los veterinarios y sus especialidades, para tener un control del personal de la clínica.  
Fuente: VetController.java
