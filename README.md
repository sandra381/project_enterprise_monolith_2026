# Spring PetClinic - DevEx Audit
Este repositorio contiene la documentación del análisis realizado sobre el proyecto Spring PetClinic.

## Entorno con un solo comando

Asegúrate de tener Docker y Docker Compose instalados. Luego ejecuta:

```bash
docker-compose up
```
